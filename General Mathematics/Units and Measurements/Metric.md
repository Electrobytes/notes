The metric system inherits the scientific notation to simplify the reading of a large value. The prefix of a unit corresponds to an exponent found in the scientific notation. The table below are the commonly used prefixes, but the full list can be found [[Metric Prefixes|here]].

| Prefix | Symbol | Exponent | Scientific Notation | Standard Notation |
| ------ | ------ | -------- | ------------------- | ----------------- |
| kilo-  | k      | 3        | $*10^3$             | 1000              |
| *none* |        |          | $*10^0$             | 1                 |
| centi- | c      | -2       | $*10^{-2}$          | 0.01              |
| milli  | m      | -3       | $*10^{-3}$          | 0.001             |
